# DrinkMate - 合上圆环饮酒版 产品需求文档

> 一个结合 Apple Watch 圆环交互理念的饮酒记录与科学饮酒指导应用

> 文档版本：v1.0 | 创建日期：2026-04-20

## 一、产品概述

### 1.1 产品定位
DrinkMate 是一款面向注重健康饮酒用户的移动端 Web 应用，通过"合上圆环"的游戏化机制，帮助用户记录饮酒、换算酒精度数、指导科学饮酒。

### 1.2 目标用户
- 25-40岁都市白领
- 有社交饮酒需求但关注健康
- 已习惯使用 Apple Watch 或类似健康应用
- 希望量化控制饮酒量

### 1.3 核心价值主张
> "合上圆环" = 完成今日健康饮酒目标

## 二、核心功能模块

### 2.1 圆环进度系统 ⭐核心卖点

#### 视觉设计
| 元素 | 说明 |
|------|------|
| 外环 | 进度环，根据摄入量动态填充 |
| 中环 | 刻度装饰，增加精致感 |
| 内环 | 背景，显示剩余目标量 |

#### 配色系统
| 进度区间 | 颜色 | 含义 |
|----------|------|------|
| 0-50% | 金色渐变 `#d4a853 → #b8944a` | 安全区，继续保持 |
| 50-80% | 琥珀渐变 `#fbbf24 → #f59e0b` | 警告区，注意控制 |
| 80-100% | 橙色渐变 `#f97316 → #ea580c` | 临界区，即将超量 |
| 100%+ | 红色渐变 `#f87171 → #dc2626` | 危险区，已超上限 |

#### 计算公式
```
纯酒精(g) = 容量(ml) × 酒精度(%) × 0.8(密度)

标准杯换算 = 纯酒精(g) ÷ 10
      （中国标准：1标准杯 = 10g纯酒精）

热量(kcal) = 纯酒精(g) × 7
      （每克酒精7千卡）
```

#### 安全阈值
| 性别 | 每日上限 | 每周建议 |
|------|---------|---------|
| 男性 | 25g纯酒精 (2.5标准杯) | 至少2-3天不饮酒 |
| 女性 | 15g纯酒精 (1.5标准杯) | 至少2-3天不饮酒 |

### 2.2 饮酒记录系统

```javascript
// 记录数据结构
interface DrinkRecord {
  id: number;           // 时间戳
  type: DrinkType;      // 'wine' | 'beer' | 'spirit' | 'cocktail' | 'sake' | 'whiskey'
  name: string;         // 饮品名称
  volume: number;       // 容量 ml
  abv: number;          // 酒精度 %
  time: string;         // ISO 8601 时间
  pureAlcohol: number;  // 计算得出的纯酒精 g
}
```

**录入方式：**
- **快速录入**（FAB按钮）：直接打开表单，手动填写
- **一键录入**（快捷按钮）：点击啤酒/葡萄酒/烈酒/鸡尾酒图标，自动填充默认值
- **智能推荐**：根据时间推荐（午餐→啤酒，晚餐→葡萄酒，深夜→烈酒）

**预设值配置：**
| 类型 | 图标 | 默认名称 | 默认酒精度 | 默认容量 |
|------|------|---------|-----------|---------|
| 葡萄酒 | 🍷 | 葡萄酒 | 12% | 150ml |
| 啤酒 | 🍺 | 啤酒 | 4% | 500ml |
| 白酒/烈酒 | 🥃 | 白酒 | 52% | 50ml |
| 鸡尾酒 | 🍸 | 鸡尾酒 | 15% | 100ml |
| 清酒 | 🍶 | 清酒 | 15% | 100ml |
| 威士忌 | 🥃 | 威士忌 | 40% | 45ml |

### 2.4 科学饮酒指导系统

**建议卡片内容：**

**卡片 1：适度标准**
```
🟢 适度标准

男性每日 ≤ 25g 纯酒精 (约 2.5 标准杯)
女性每日 ≤ 15g 纯酒精 (约 1.5 标准杯)

每周建议至少有 2-3 天完全不饮酒，
给肝脏足够的代谢和恢复时间。
```

**卡片 2：饮酒时机**
```
🟡 饮酒时机

✓ 饮酒前先吃些食物，减缓酒精吸收
✓ 边吃边喝，控制饮酒速度
✗ 避免空腹饮酒
✗ 不与碳酸饮料混饮（加速吸收）
✓ 饮酒后多喝水，促进代谢
```

**卡片 3：安全提醒**
```
🔴 安全提醒

⚠️ 服用药物期间绝对禁酒
   （尤其抗生素、镇静剂、降糖药）

⚠️ 备孕、怀孕、哺乳期完全禁酒

🚫 酒后绝对禁止驾车
   （即使"感觉清醒"也可能超标）
```

**动态建议逻辑：**
| 当前进度 | 圆环颜色 | 提示文案 |
|---------|---------|---------|
| 0% | 金色 | "今日尚未饮酒，保持清醒也是好选择" |
| 1-50% | 金色 | "适量范围，继续保持" |
| 51-80% | 琥珀 | "接近上限，请注意控制" |
| 81-99% | 橙色 | "即将达到上限，建议停止" |
| 100%+ | 红色 | "已超安全上限，请立即停止饮酒" |
